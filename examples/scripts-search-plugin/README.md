# Rusty Scripts Search Plugin

汎用プラグインのサンプルです。
オブジェクト・エフェクトを検索し、タイムラインに配置する機能を提供します。

> [!TIP]
> フィルタスクリプトの配置時はShift/Altキーを押すと挙動が変わります：
> - 通常クリック：フィルタ効果として配置
> - Shift+クリック：選択中のオブジェクトにエフェクトとして追加
> - Alt+クリック：フィルタオブジェクトとして配置

## インストール

`C:\ProgramData\aviutl2\Plugin` に `rusty_scripts_search.aux2` を配置してください。
