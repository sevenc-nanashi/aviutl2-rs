# Rusty FFmpeg Output

出力プラグインのサンプルです。

- FFmpegを使用して動画を出力する
- eguiを使用してFFmpegのオプションを設定する

推奨される代替：[rigaya/ffmpegOut](https://github.com/rigaya/ffmpegOut)

## 比較

- プロジェクト：[dvd_for_5_minutes.aup2](../../projects/dvd_for_5_minutes.aup2)
- 計測方法：Microsoft Game DVRの録画を使用。
    - タイマースタート：下のバーが赤くなってから
    - タイマーストップ：下のバーが青になるまで
- フォーマット：mp4

- CPU：i7-1360P、RAM：16GB、OS：Windows 11 Pro、書き出し先：SoftPerfect RAM Disk（1GB）

    - Rusty FFmpeg Output：2m05s、72.0fps
